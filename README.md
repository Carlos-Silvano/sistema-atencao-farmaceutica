Sistema de Gestão para Atenção Farmacêutica
Este projeto consiste em um sistema web desenvolvido para otimizar o acompanhamento e a gestão de cuidados farmacêuticos. O foco principal é a organização de dados de pacientes e medicamentos, garantindo segurança e eficiência no atendimento

Tecnologias Utilizadas
Linguagem: Python.

Framework Web: Flask.

Banco de Dados: SQLite.

Frontend: HTML5, CSS3 e Jinja2.

Funcionalidades
Cadastro e gerenciamento de perfis de pacientes.

Registro de intervenções farmacêuticas e uso de medicamentos.

Interface responsiva para facilitar o uso em diferentes dispositivos.
